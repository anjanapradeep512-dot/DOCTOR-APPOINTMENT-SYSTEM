# Doctor Appointment Management System

## Overview

The Doctor Appointment Management System is a web-based application designed to simplify and automate the process of scheduling medical appointments between patients and doctors. The system provides an efficient, secure, and user-friendly platform for managing appointments, reducing the challenges associated with manual scheduling.

Patients can register, search for doctors based on specialization, view available appointment slots, and book appointments online. Doctors and administrators can manage schedules, monitor appointments, and maintain healthcare records efficiently.

## Features

### Patient Module

* Patient Registration and Login
* Search Doctors by Specialization
* View Doctor Availability
* Book Appointments
* Cancel Appointments
* View Appointment History

### Doctor Module

* Manage Appointment Schedule
* View Upcoming Appointments
* Monitor Patient Bookings

### Admin Module

* Manage Doctors and Patients
* View and Manage Appointments
* Generate Reports
* Maintain System Records

## Database Design

The system maintains separate databases/tables for:

* Patients
* Doctors
* Appointments
* Administrators

The project demonstrates key Database Management System (DBMS) concepts including:

* Database Normalization
* Relational Database Design
* Entity Relationships
* SQL Queries
* Data Integrity
* Transaction Management

## System Workflow

1. Patient registers and logs into the system.
2. Patient searches for doctors based on specialization.
3. Available appointment slots are displayed.
4. Patient books a suitable appointment.
5. Appointment details are stored in the database.
6. Doctors and administrators can view and manage appointments.
7. Patients can view appointment history or cancel appointments when required.
